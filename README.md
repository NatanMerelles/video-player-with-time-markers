um video player desenvolvido em Meteor, com o uso de sua biblioteca front-end, Blaze.
o projeto foi inspirado no componente React chamado react-player, servindo como forma de entender as semelhanças e diferenças do lifecycle de ambas tecnologias. 

Além disso, o projeto conta com uma feature a mais, que são os markers de tempo com comentarios, permitindo anotações durante a visualização do video.
